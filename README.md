# GYA: GESTIÓN INTELIGENTE DE INVENTARIOS FARMACÉUTICOS

## DESCRIPCIÓN: 
Este proyecto desarrolla una arquitectura de datos integrada que utiliza procesos ETL (Extracción, Transformación y Carga) para centralizar información fragmentada de sistemas ERP, HIS y registros manuales. El objetivo es transformar la gestión reactiva de inventarios farmacéuticos en un modelo de Big Data 2.0, permitiendo la mitigación proactiva de riesgos de desabastecimiento y vencimiento de medicamentos mediante indicadores confiables y trazabilidad en tiempo real.
